# viewnsxiv

### Requirements
- `sxiv`
- `curl`

## Installation

```
git clone https://github.com/MyOS-ArchLinux/viewnsxiv
cd viewnsxiv/
sudo install -m755 viewnsxiv /usr/local/bin/viewnsxiv
```

## Usage
```
viewnsxiv https://avatars.githubusercontent.com/u/80172301
viewnsxiv ./test.png
viewnsxiv ./dir/ 
```
